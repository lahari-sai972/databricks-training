# WINDOW FUNCTIONS IN MYSQL – SQL PRACTICE

In this project, I practiced MySQL Window Functions using employee data.

---

# Functions Practiced

* `ROW_NUMBER()`
* `RANK()`
* `DENSE_RANK()`
* `OVER()`
* `PARTITION BY`
* `ORDER BY`

---

# Table Used

## employees

Contains employee details such as:

- Employee ID
- Employee Name
- Department
- Salary
- Joining Date

---

# Questions Covered

## ROW_NUMBER()

1. Assign row numbers based on highest salary
2. Assign row numbers within department based on salary descending
3. Assign row numbers based on latest joining date
4. Assign row numbers within department based on earliest joining date
5. Assign row numbers based on salary ascending
6. Assign row numbers within department based on employee name alphabetically

---

## RANK()

7. Rank employees based on highest salary
8. Rank employees within department based on salary
9. Rank employees based on latest joining date
10. Rank employees based on employee name alphabetically
11. Rank employees within department based on lowest salary

---

## DENSE_RANK()

12. Assign dense rank based on highest salary
13. Assign dense rank within department based on salary
14. Assign dense rank based on joining date
15. Assign dense rank based on salary ascending
16. Assign dense rank within department based on joining date
